# Builder API

The Builder API is the primary Go-native way to create PDFs. It uses a fluent builder pattern with nested closures: **Document -> Page -> Row -> Col -> Element**.

## Creating a Document

```go
import (
    "github.com/gpdf-dev/gpdf/document"
    "github.com/gpdf-dev/gpdf/template"
)

doc := template.New(
    template.WithPageSize(document.A4),
    template.WithMargins(document.UniformEdges(document.Mm(20))),
)
```

### Document Options

| Option | Description | Default |
|---|---|---|
| `WithPageSize(size)` | Set page dimensions | `document.A4` |
| `WithMargins(edges)` | Set page margins | `Mm(20)` uniform |
| `WithFont(family, data)` | Register a TrueType font | -- |
| `WithDefaultFont(family, size)` | Set default font family and size | `""`, `12` |
| `WithMetadata(meta)` | Set PDF metadata (title, author, etc.) | -- |

### Document Methods

| Method | Description |
|---|---|
| `doc.AddPage()` | Add a new page, returns `*PageBuilder` |
| `doc.Header(fn)` | Register header builder (repeated on every page) |
| `doc.Footer(fn)` | Register footer builder (repeated on every page) |
| `doc.Generate()` | Produce PDF as `[]byte` |
| `doc.Render(w)` | Write PDF to an `io.Writer` |

## Pages

```go
page := doc.AddPage()
```

Each call to `AddPage()` creates a new page. Content that overflows a page is automatically paginated to subsequent pages.

### Rows

Rows are the horizontal containers within a page. There are two types:

```go
// Auto-height: row height is determined by content
page.AutoRow(func(r *template.RowBuilder) {
    // ...
})

// Fixed-height: row has an explicit height
page.Row(document.Mm(30), func(r *template.RowBuilder) {
    // ...
})
```

### Columns

Within each row, columns are arranged using the 12-column grid:

```go
page.AutoRow(func(r *template.RowBuilder) {
    r.Col(8, func(c *template.ColBuilder) {
        c.Text("Wide column (8/12)")
    })
    r.Col(4, func(c *template.ColBuilder) {
        c.Text("Narrow column (4/12)")
    })
})
```

The `span` parameter (1-12) determines the column's width as a fraction of the total row width. Column spans in a row should sum to 12.

## Column Elements

The `ColBuilder` provides methods for adding content elements:

| Method | Description |
|---|---|
| `c.Text(text, opts...)` | Add a text block |
| `c.Image(data, opts...)` | Add an image from `[]byte` |
| `c.Table(header, rows, opts...)` | Add a data table |
| `c.List(items, opts...)` | Add an unordered (bullet) list |
| `c.OrderedList(items, opts...)` | Add an ordered (numbered) list |
| `c.Line(opts...)` | Add a horizontal line |
| `c.Spacer(height)` | Add vertical spacing |
| `c.QRCode(data, opts...)` | Add a QR code |
| `c.Barcode(data, opts...)` | Add a barcode |
| `c.PageNumber(opts...)` | Insert current page number |
| `c.TotalPages(opts...)` | Insert total page count |
| `c.RichText(fn, opts...)` | Add mixed-style inline text |

## Complete Example

```go
doc := template.New(
    template.WithPageSize(document.A4),
    template.WithMargins(document.UniformEdges(document.Mm(20))),
    template.WithMetadata(document.DocumentMetadata{
        Title:  "Sales Report",
        Author: "ACME Corp",
    }),
)

// Header (appears on every page)
doc.Header(func(p *template.PageBuilder) {
    p.AutoRow(func(r *template.RowBuilder) {
        r.Col(6, func(c *template.ColBuilder) {
            c.Text("ACME Corporation", template.Bold(), template.FontSize(10))
        })
        r.Col(6, func(c *template.ColBuilder) {
            c.Text("Confidential", template.AlignRight(), template.FontSize(10),
                template.TextColor(pdf.Gray(0.5)))
        })
    })
    p.AutoRow(func(r *template.RowBuilder) {
        r.Col(12, func(c *template.ColBuilder) {
            c.Line(template.LineColor(pdf.RGBHex(0x1565C0)))
            c.Spacer(document.Mm(5))
        })
    })
})

// Footer (appears on every page)
doc.Footer(func(p *template.PageBuilder) {
    p.AutoRow(func(r *template.RowBuilder) {
        r.Col(6, func(c *template.ColBuilder) {
            c.Text("Generated by gpdf", template.FontSize(8),
                template.TextColor(pdf.Gray(0.5)))
        })
        r.Col(6, func(c *template.ColBuilder) {
            c.PageNumber(template.AlignRight(), template.FontSize(8),
                template.TextColor(pdf.Gray(0.5)))
        })
    })
})

// Page content
page := doc.AddPage()

page.AutoRow(func(r *template.RowBuilder) {
    r.Col(12, func(c *template.ColBuilder) {
        c.Text("Sales Report - Q1 2026", template.FontSize(24), template.Bold())
        c.Spacer(document.Mm(10))
    })
})

page.AutoRow(func(r *template.RowBuilder) {
    r.Col(12, func(c *template.ColBuilder) {
        c.Table(
            []string{"Product", "Units Sold", "Revenue"},
            [][]string{
                {"Widget A", "1,200", "$24,000"},
                {"Widget B", "850", "$42,500"},
                {"Widget C", "2,100", "$31,500"},
            },
            template.ColumnWidths(40, 30, 30),
            template.TableHeaderStyle(
                template.TextColor(pdf.White),
                template.BgColor(pdf.RGBHex(0x1A237E)),
            ),
            template.TableStripe(pdf.RGBHex(0xF5F5F5)),
        )
    })
})

data, err := doc.Generate()
```

## Writing to an `io.Writer`

Instead of getting a `[]byte`, you can write directly to any `io.Writer`:

```go
// Write to a file
f, _ := os.Create("output.pdf")
defer f.Close()
doc.Render(f)

// Write to an HTTP response
func handler(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "application/pdf")
    doc.Render(w)
}

// Write to a buffer
var buf bytes.Buffer
doc.Render(&buf)
```
